# Riwayat Perubahan (Changelog)

Semua perubahan penting pada proyek **NestCraft Network Landing Page** akan dicatat di file ini secara teratur.

Format riwayat perubahan ini didasarkan pada [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) dan proyek ini mematuhi [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.1.0] - 2026-07-04

### Ditambahkan
- **Cinematic Background**: Latar belakang berkualitas tinggi dengan nuansa digital modern Minecraft di Hero Section (`/public/hero-bg.jpg`).
- **Portfolio Link Showcase**: Menambahkan tombol dan tautan eksternal ke [Katalog Web Server Minecraft RAN DEV](https://sfl.gl/x2ic) di bagian Top Header Bar dan Footer.
- **GitHub Templates**: Menambahkan dokumen `/CONTRIBUTING.md`, `/CODE_OF_CONDUCT.md`, `/SECURITY.md`, `/CHANGELOG.md`, `.editorconfig`, dan `.gitattributes`.
- **Master Prompt Universal**: Menambahkan `/MASTER_PROMPT.md` yang super lengkap dan detail untuk meng-generate landing page Minecraft server masa depan secara otomatis.

### Diubah
- Mengoptimalkan transparansi logo dan tata letak grid red neon agar menyatu sempurna dengan gambar background baru.

---

## [1.0.0] - 2026-07-01

### Ditambahkan
- **Hero Section**: Dilengkapi logo menyala, judul besar, dan tombol CTA.
- **Game Modes**: Grid berisi 6 kartu mode game: Survival, PvP, OneBlock, SkyBlock, Creative, dan Parkour.
- **Server Info Section**: Menampilkan IP Java & Bedrock serta Port Bedrock lengkap dengan status server *real-time* via API publik, tombol salin instan, dan panduan bermain.
- **Discord CTA Box**: Tombol besar untuk bergabung ke server komunitas Discord.
- **Developer Watermark**: Kredit untuk **RAN DEV** di Top Header Bar dan Footer lengkap dengan nomor WhatsApp.
- **SEO & PWA Optimization**: Integrasi file `robots.txt`, `sitemap.xml`, `manifest.webmanifest`, Meta Tag, Open Graph, dan Schema JSON-LD.
