# Kebijakan Keamanan (Security Policy)

Kami sangat berkomitmen untuk menjaga keamanan data, privasi, serta keandalan aplikasi **NestCraft Network Landing Page**. Jika Anda menemukan adanya celah keamanan atau kerentanan dalam proyek ini, silakan ikuti panduan berikut untuk melaporkannya secara aman.

---

## Versi yang Didukung

Kami sangat menyarankan Anda untuk selalu menggunakan versi stabil terbaru dari kode sumber kami di branch `main`. Perbaikan keamanan hanya akan diterapkan pada versi-versi berikut:

| Versi | Didukung |
| :--- | :---: |
| v1.0.x (Terbaru) |  Ya |
| < v1.0.0 | Tidak |

---

2. Tuliskan detail celah keamanan yang ditemukan secara jelas, termasuk:
